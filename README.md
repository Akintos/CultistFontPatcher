# CultistFontPatcher
[Cultist Simulator](https://store.steampowered.com/app/718670/Cultist_Simulator/) Korean font patcher.

[컬티스트 시뮬레이터](https://store.steampowered.com/app/718670/Cultist_Simulator/) 한글 폰트 패치 프로그램. 구글의 [Noto Serif CJK KR Regular](https://www.google.com/get/noto/#serif-kore) 폰트를 사용합니다

## 적용방법
Samaritan님의 [한국어 패치 배포 페이지](https://steamcommunity.com/app/718670/discussions/1/2525904966948106423/) 에서 패치된 애셋 파일과 한국어 번역 데이터를 제공합니다.

직접 애셋을 패치하고자 하시는 분은 [releases](../../releases) 에서 최신 패치 프로그램을 받아서 게임 설치폴더에 넣고 실행하시면 됩니다.
